# 🛒 MarkeTech

Tienda virtual de tecnología desarrollada por aprendices ADSO del SENA CTM Itagüí.

---

## 📋 Descripción

MarkeTech es una tienda virtual orientada a la comercialización de productos tecnológicos para estudiantes, desarrolladores y profesionales en Colombia.

El proyecto fue desarrollado como ejercicio académico aplicando buenas prácticas de desarrollo web, organización de código y despliegue mediante GitHub Pages.

---

## 🎯 Objetivos del proyecto

* Promover la venta de productos tecnológicos.
* Implementar una interfaz moderna y responsiva.
* Aplicar conocimientos de HTML5, CSS3 y control de versiones con Git.
* Publicar una aplicación web estática utilizando GitHub Pages.

---

## 🛠️ Tecnologías utilizadas

| Tecnología   | Propósito                |

Para facilitar el mantenimiento del proyecto se definieron las siguientes convenciones:

## 📚 Control de versiones

Se utiliza Git para registrar cambios y facilitar el trabajo colaborativo entre los integrantes del proyecto.

Tipos de commits utilizados:

* feat: nuevas funcionalidades
* fix: corrección de errores
* docs: documentación
* style: cambios visuales
* refactor: mejoras de estructura interna
