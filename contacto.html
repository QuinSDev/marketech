Preguntas Frecuentes

¿Qué es MERKATECH?

MERKATECH es un proyecto académico desarrollado por estudiantes de Análisis y Desarrollo de Software (ADSO), enfocado en la creación de una plataforma web para la visualización y gestión de productos tecnológicos.

¿Los productos publicados están disponibles para compra real?

No. Los productos mostrados en esta plataforma tienen fines educativos y demostrativos. El objetivo es mostrar el diseño, organización y funcionalidades de una tienda tecnológica en línea.

¿Qué tecnologías se utilizaron para desarrollar el proyecto?

El proyecto fue desarrollado utilizando HTML5, CSS3, JavaScript, Git y GitHub Pages, aplicando buenas prácticas de diseño web responsivo y control de versiones.

¿Quiénes desarrollaron MERKATECH?

El proyecto fue desarrollado por Andrés Gutiérrez y Sebastián Quintero como parte de su proceso de formación en el programa de Análisis y Desarrollo de Software (ADSO).

¿Cómo puedo reportar un error o enviar sugerencias?

Puedes utilizar el formulario de contacto disponible en esta página. Todas las observaciones y sugerencias ayudan a mejorar el proyecto y fortalecer la experiencia del usuario.
